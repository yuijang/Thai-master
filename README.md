# Metro Lotto - 한국 로또 추첨기

메트로 UI(Metro UI) 스타일로 디자인된 한국 로또(6/45) 번호 추첨기 웹사이트입니다.

## 주요 기능
- **메트로 디자인:** Windows 스타일의 플랫하고 현대적인 타일 레이아웃을 사용합니다.
- **로또 번호 생성:** 1~45 사이의 중복되지 않는 6개 숫자를 랜덤으로 추출합니다.
- **색상 구분:** 한국 로또 공식 번호 대역별 색상을 적용했습니다.
  - 1-10: 노란색
  - 11-20: 파란색
  - 21-30: 빨간색
  - 31-40: 회색
  - 41-45: 녹색
- **애니메이션:** 번호가 순차적으로 나타나는 부드러운 애니메이션 효과를 제공합니다.

## 배포 방법 (GitHub Pages)
1. GitHub 저장소 설정(Settings)으로 이동합니다.
2. 왼쪽 메뉴에서 **Pages**를 클릭합니다.
3. **Build and deployment** 섹션의 **Branch**를 `main`으로 선택하고 **Save**를 클릭합니다.
4. 잠시 후 상단에 표시되는 URL을 통해 접속할 수 있습니다.
